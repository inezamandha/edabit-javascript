# Even All the Way

`Arrays` `Language Fundamentals` `Numbers`

[View on Edabit](https://edabit.com/challenge/6MZx5RqKYkFaogeAQ)

---

Given an array of numbers, return an array which contains all the even numbers in the original array, which also have even _indices_.

### **Examples**

```
getOnlyEvens([1, 3, 2, 6, 4, 8]) ➞ [2, 4]

getOnlyEvens([0, 1, 2, 3, 4]) ➞ [0, 2, 4]

getOnlyEvens([1, 2, 3, 4, 5]) ➞ []
```

### **Notes**

Arrays start at index 0.
