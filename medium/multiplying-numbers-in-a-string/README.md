# Multiplying Numbers in a String

`Language Fundamentals` `Numbers` `Strings`

[View on Edabit](https://edabit.com/challenge/TzxoYExuxuQw2ahWR)

---

Given a string of numbers separated by a _comma and space_, return the product of the numbers.

### **Examples**

```
multiplyNums("2, 3") ➞ 6

multiplyNums("1, 2, 3, 4") ➞ 24

multiplyNums("54, 75, 453, 0") ➞ 0

multiplyNums("10, -2") ➞ -20
```

### **Notes**

Bonus: Try to complete this challenge in **one line**!
