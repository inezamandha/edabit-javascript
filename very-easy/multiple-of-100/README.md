# Multiple of 100

`Algebra` `Conditions` `Math` `Validation`

[View on Edabit](https://edabit.com/challenge/qMr6wYGr6NaXAPQGF)

---

Create a function that takes an integer and returns `true` if it's divisible by 100, otherwise return `false`.

### **Examples**

```
divisible(1) ➞ false

divisible(1000) ➞ true

divisible(100) ➞ true
```
