# Is the Object Empty?

`Language Fundamentals` `Objects` `Validation`

[View on Edabit](https://edabit.com/challenge/nGybgqB9agSqvBszF)

---

Write a function that returns `true` if an object is empty, and `false` otherwise.

### **Examples**

```
isEmpty({}) ➞ true

isEmpty({ a: 1 }) ➞ false
```
