# Char-to-ASCII

`Algorithms` `Numbers` `Strings`

[View on Edabit](https://edabit.com/challenge/xPAPYC24EEpwzwQDy)

---

Create a function that returns the ASCII value of the passed in character.

### **Examples**

```
ctoa("A") ➞ 65

ctoa("m") ➞ 109

ctoa("[") ➞ 91

ctoa("\") ➞ 92
```
