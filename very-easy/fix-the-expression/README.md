# Fix the Expression

`Bugs` `Language Fundamentals` `Validation`

[View on Edabit](https://edabit.com/challenge/FipbQSYquQLPZ8QXG)

---

Fix the code in the **Code** tab so the function returns `true` _if and only if_ `x` is equal to `7`.

### **Examples**

```
isSeven(4) ➞ false

isSeven(9) ➞ false

isSeven(7) ➞ true
```

### **Notes**

The bug can be hard to find, so look closely!
