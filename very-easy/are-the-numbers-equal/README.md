# Are the Numbers Equal?

`Conditions` `Language Fundamentals` `Numbers` `Validation`

[View on Edabit](https://edabit.com/challenge/QSnaSH5S3oxZkwcNc)

---

Create a function that returns `true` when `num1` is equal to `num2`; otherwise return `false`.

### **Examples**

```
isSameNum(4, 8) ➞ false

isSameNum(2, 2) ➞  true

isSameNum(2, "2") ➞ false
```

### **Notes**

Don't forget to `return` the result.
