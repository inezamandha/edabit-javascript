# Re-Form the Word

`Formatting` `Language Fundamentals` `Strings`

[View on Edabit](https://edabit.com/challenge/KQQ7RMXjSK257GQgy)

---

A word has been split into a `left` part and a `right` part. Re-form the word by adding both halves together, changing the _first_ character to an uppercase letter.

### **Examples**

```
getWord("seas", "onal") ➞ "Seasonal"

getWord("comp", "lete") ➞ "Complete"

getWord("lang", "uage") ➞ "Language"
```
