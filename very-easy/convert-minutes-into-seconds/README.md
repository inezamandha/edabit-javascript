# Convert Minutes into Seconds

`Language Fundamentals` `Math` `Numbers`

[View on Edabit](https://edabit.com/challenge/8q54MKnRrm89pSLmW)

---

Write a function that takes an integer `minutes` and converts it to seconds.

### **Examples**

```
convert(5) ➞ 300

convert(3) ➞ 180

convert(2) ➞ 120
```
